# python-flask-api
flaskでapiを実装

## 技術スタック
- Python 3.8+
- Flask (Webフレームワーク)
- SQLAlchemy (ORM)

```
│       └── main.py
├── README.md
├── requirements-dev.txt
└── requirements.txt
```
